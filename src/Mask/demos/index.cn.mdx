#### 简介

背景蒙版，用于Modal等弹出层。蒙版出现时，body不可滚动，下方内容可见不可点击。


## 开关

通过设置 open 属性开关蒙版。

```javascript
import React, { useState } from 'react';
import Mask from '../index';

const Demo = () => {
    const [open, set] = useState(false);
    const handleClick = () => {
        set(true);
        setTimeout(() => set(false), 3000);
    }
    return (
        <>
            <button onClick={handleClick}>点击打开蒙版，三秒后关闭</button>
            <Mask open={open}/>
        </>
    );
}
```

## API

| 属性 | 说明     | 类型                                         | 默认值 | 
| ---- | -------- | -------------------------------------------- | ------ |
| open | 开关蒙版 | boolean |
| zIndex | 蒙版层级 | number | 100 |
| 其他 | html属性 | 如 class, style, onClick 等 | |
